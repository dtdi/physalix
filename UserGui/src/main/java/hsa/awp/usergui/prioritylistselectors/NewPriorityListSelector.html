Wie erstelle ich meine Wunschlisten?

  • Wenn Sie eine Veranstaltung besuchen wollen, dann erstellen Sie wie folgt eine Wunschliste:

    Ziehen Sie mit der Maus Ihre gewünschten Veranstaltungen nach Ihrer persönlichen Priorität sortiert in die Wunschliste.

  • Wenn Sie mehr als eine Veranstaltung besuchen wollen, dann erstellen Sie eine zweite Wunschliste oder auch noch weitere. Allerdings ist die maximal mögliche Anzahl von Wunschlisten für jede Einschreibung begrenzt. Sie können insgesamt höchstens so viele Wunschlisten ausfüllen, wie Sie in der Anzeige sehen.

Kann ich meine Wunschliste später noch verändern?
Wann habe ich fest gebucht?

  • Während der Einschreibephase können Sie Ihre Wunschlisten jederzeit verändern.

    Wählen Sie dazu wieder die Option "Anmelden" und danach die entsprechende Anmeldemöglichkeit.

  • Am Ende der der Einschreibephase findet die Buchung durch eine Verlosung der verfügbaren Plätze statt.

  • Über das Ergebnis der Verlosung werden Sie per Mail informiert.

  • Die Buchungen können Sie dann über die Auswahl "Buchungen verwalten" einsehen und bearbeiten.

Help

How do I set up my course preference list?

  • To register for a specific course set up a preference list in the following manner:

    Drag and drop all desired courses into a preference list according to your desired priority.

  • If you would like to attend more than one language course, then set up an additional preference list, or as many as needed. However, the total number of preference lists for each student enrolled is limited. It is only possible to set up the maximum number of preference lists as shown in the window.

Can I make changes to my preference list once it is set up?
When is my course enrollment binding?

  • During the registration period you are able to make changes to your preference lists at any point in time.

    Click the option "Register" and then click on the relevant registration request.

  • At the end of the registration period, final course enrollment will be determined by means of a lottery for all students.

  • The results of the course lottery will be sent to your e-mail address.

  • To view or make changes to enrollment in a specific course, click on the button "Course Enrollment Options".
